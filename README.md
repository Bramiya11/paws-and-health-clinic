# 🐾 Paws And Health Clinic
> Forked from: [Original Repository Link](https://github.com/DanielDev87/actu-mant-soft-26-1)


## 📖 Overview
This project is a **Pet Store and Veterinary Clinic Management System** designed to simulate a real-world application used to manage pet products, veterinary services, and client information.

The repository is used for **software maintenance and evolution practice**, including debugging, refactoring, feature updates, and system improvements.

> This repository is a **fork of the original project** and is used for academic purposes related to software maintenance and version control workflows.

## 🎯 Objectives
The main goal of this project is to practice key concepts of **Software Maintenance and Evolution**, such as:

- Bug fixing
- Code refactoring
- Feature enhancements
- System updates
- Version control workflows
- Code quality improvements

## 🛠️ Technologies
This project may include technologies such as:

- Programming Language: *JavaScript/EJS*
- Database: *MySQL*
- Version Control: Git & GitHub

## 🐶 System Features
The system simulates the operations of a **Pet Store and Veterinary Clinic**, including:

- Veterinary service management
- Customer information management
- Appointment or service records

## 🔧 Maintenance Activities
As part of the maintenance process, the following activities may be performed:

- Corrective maintenance (bug fixes)
- Adaptive maintenance (system updates)
- Perfective maintenance (performance or usability improvements)
- Preventive maintenance (code refactoring and optimization)

## 🌿 Git Workflow
This repository is used to practice **Git and GitHub workflows**, including:

- Forking repositories
- Creating branches
- Making commits
- Pull requests
- Tracking changes through version history

## 🎓 Educational Purpose
This project is part of an academic exercise focused on understanding how software systems evolve after their initial development.
